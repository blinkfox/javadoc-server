Javadoc 服务

这是一个用来托管和查看 Javadoc 的 SpringBoot 服务。当你输入 Maven 坐标时，会自动从 Maven 中央仓库拉取 Javadoc 资源，并存储到 MinIO 中，方便大家查看和使用。

尝试一下

请在下方的输入框中输入你需要查看的 javadoc.jar 包的 groupId、artifactId、和version，然后"点击查看"即可。

groupId
artifactId
version
注意：如果该 javadoc.jar 是第一次被加载访问的话，需要等候一段时间。

使用概述

  • 只需要通过 URL 即可访问你需要的 javadoc 资源，如：/docs/{groupId}/{artifactId}/{version}
